Wo soll das Teach-Panel erscheinen?

Die aktuelle App hat links die Canvas (Matte), rechts eine schmale Sidebar mit Einstellungen.

Canvas / Matte
Settings
Sidebar
⚙ Teach Panel (schwebend)

A — Schwebende Palette

Kleines Fenster das frei auf dem Canvas schweben kann. Kann verschoben werden. Immer sichtbar, überlappend.

Canvas / Matte
Settings
Sidebar
↕ Teach Panel (ausklappbar unten)

B — Unteres Panel

Klappt am unteren Rand auf. Canvas bleibt sichtbar. Gut für breite Layouts (Jog-Buttons nebeneinander).

Canvas / Matte
⚙ Teach
Panel
(ersetzt Sidebar)

C — Sidebar ersetzen

Im Teach-Modus ersetzt das Panel die rechte Sidebar vollständig. Umschalten über Toolbar-Button.

Klick auf eine Option — oder schreib mir direkt im Terminal.
